2D / 3D 物理碰撞几何体与检测算法

物理碰撞的核心是用简单几何体近似复杂模型，并判断两个几何体是否相交。本文的计算口径是：碰撞体有本地形状，物体 Transform 只考虑位置和旋转，不考虑缩放。检测前先把 Local Collider 转成 World Collider，再进入宽阶段和窄阶段。

总览

游戏物理不会直接拿复杂渲染网格做所有碰撞。渲染模型可能有几万三角形，但物理通常用更简单的碰撞体，例如圆、盒子、球、胶囊、凸包。这样速度更快，也更稳定。

碰撞检测流程

完整碰撞系统不应该每帧把所有物体两两检测。假设有 N 个物体，暴力检测是 O(N^2)，物体一多就会很慢。

Brute force pair count = N * (N - 1) / 2
  1. 每个物体先把本地碰撞体按 position + rotation 转到世界空间。
  2. 根据世界空间碰撞体重新计算宽阶段 AABB。
  3. 用空间结构找出可能重叠的 AABB 对。
  4. 对每个候选对调用具体几何体的窄阶段检测。
  5. 如果碰撞，计算 normal、depth、contact point。
  6. 把接触信息交给物理解算器。

先做学习版引擎时，建议从 AABB 宽阶段 + 圆/球/AABB/胶囊窄阶段开始。SAT、GJK、EPA 放在基础系统稳定后再加。

旋转与世界碰撞体

本文后续所有检测默认都发生在世界空间。物体的缩放不参与碰撞体计算，只有平移和旋转参与。这样公式更稳定，也更接近很多游戏里“物理体尺寸固定、模型视觉可缩放但碰撞不缩放”的做法。

2D 旋转

R(a) = [ cos(a) -sin(a) ] [ sin(a) cos(a) ] worldPoint = position + R(rotation) * localPoint worldDirection = R(rotation) * localDirection

3D 旋转

3D 中可以用四元数或 3 个正交轴表示旋转。下面公式使用物体的世界方向轴：

axisX = rotation * (1, 0, 0) axisY = rotation * (0, 1, 0) axisZ = rotation * (0, 0, 1) worldPoint = position + axisX * localPoint.x + axisY * localPoint.y + axisZ * localPoint.z

各碰撞体如何受旋转影响

碰撞体 旋转后的世界数据 备注
Circle / Sphere worldCenter = position + R * localCenter，半径不变。 圆和球本身旋转后形状不变，只有本地中心偏移会被旋转。
Segment worldA = position + R * localA，worldB = position + R * localB。 线段方向会随物体旋转。
AABB 本地盒旋转后不再是 AABB，应先变成 OBB，再计算包住 OBB 的世界 AABB。 世界 AABB 主要用于宽阶段；窄阶段要用 OBB 或凸多边形。
OBB worldCenter = position + R * localCenter，worldAxes = R * localAxes，halfExtents 不变。 因为不考虑缩放，半尺寸保持不变。
Capsule 两个端点按旋转和平移变换，半径不变。 角色胶囊体常用这种方式。
Convex Polygon / Hull 所有本地顶点按 position + R * localVertex 变换到世界空间。 之后再用 SAT/GJK 检测。

旋转盒生成宽阶段 AABB

本地矩形或盒子旋转后是 OBB；宽阶段仍然需要 AABB，所以要重新包围旋转后的 OBB。

2D: ux = R * (1, 0) uy = R * (0, 1) worldHalf.x = abs(ux.x) * half.x + abs(uy.x) * half.y worldHalf.y = abs(ux.y) * half.x + abs(uy.y) * half.y worldAabb.min = worldCenter - worldHalf worldAabb.max = worldCenter + worldHalf 3D: worldHalf.x = abs(axisX.x) * half.x + abs(axisY.x) * half.y + abs(axisZ.x) * half.z worldHalf.y = abs(axisX.y) * half.x + abs(axisY.y) * half.y + abs(axisZ.y) * half.z worldHalf.z = abs(axisX.z) * half.x + abs(axisY.z) * half.y + abs(axisZ.z) * half.z

不要把“旋转后的本地 AABB”继续当成窄阶段 AABB 检测。AABB 的定义就是轴对齐；旋转后它应该是 OBB。世界 AABB 只是包围它的粗略盒子，用于宽阶段筛选。

数学基础

距离平方

很多检测只需要比较距离，不需要开方。用距离平方可以更快。

dist2 = dot(a - b, a - b) collide if dist2 <= radius^2

点到线段最近点

ab = b - a t = dot(p - a, ab) / dot(ab, ab) t = clamp(t, 0, 1) closest = a + ab * t

投影区间

SAT 会把形状投影到某条轴上，得到一段区间。如果某条轴上两个区间不重叠，就一定没有碰撞。

projection = dot(point, axis) interval = [min(projections), max(projections)] overlap if maxA >= minB && maxB >= minA

圆 - AABB 最近点

圆 - 圆数学过程

如果圆有本地中心，先应用物体旋转和平移。圆的半径不受旋转影响。

worldCenterA = positionA + rotationA * localCenterA worldCenterB = positionB + rotationB * localCenterB radiusA = localRadiusA radiusB = localRadiusB
A(centerA, radiusA) B(centerB, radiusB) d = centerB - centerA collide if dot(d, d) <= (radiusA + radiusB)^2

如果需要碰撞法线和穿透深度：

distance = length(d) normal = d / distance depth = radiusA + radiusB - distance

圆 - AABB 数学过程

closest.x = clamp(circle.center.x, box.min.x, box.max.x) closest.y = clamp(circle.center.y, box.min.y, box.max.y) d = circle.center - closest collide if dot(d, d) <= radius^2

2D AABB 碰撞

AABB = Axis-Aligned Bounding Box，轴对齐包围盒。它不旋转，所以检测非常便宜。若物体有旋转，本地矩形不能直接当世界 AABB；正确做法是先生成世界 OBB，再计算包住 OBB 的世界 AABB，用于宽阶段。

AABB 重叠
AABB 分离

数学过程

两个 AABB 在 x 轴和 y 轴都重叠，才算碰撞。

overlapX = A.min.x <= B.max.x && A.max.x >= B.min.x overlapY = A.min.y <= B.max.y && A.max.y >= B.min.y collide = overlapX && overlapY

反过来看，如果任意一个方向存在间隙，就不碰撞。

separated = A.max.x < B.min.x || B.max.x < A.min.x || A.max.y < B.min.y || B.max.y < A.min.y

旋转矩形转宽阶段 AABB

worldCenter = position + R * localCenter axisX = R * (1, 0) axisY = R * (0, 1) worldHalf.x = abs(axisX.x) * half.x + abs(axisY.x) * half.y worldHalf.y = abs(axisX.y) * half.x + abs(axisY.y) * half.y worldAabb.min = worldCenter - worldHalf worldAabb.max = worldCenter + worldHalf

2D 线段检测

线段常用于射线检测、攻击判定、平台边界和多边形边。

方向叉乘

cross2(a, b) = a.x * b.y - a.y * b.x

两个线段 p + t*r 和 q + u*s 相交时：

p + t*r = q + u*s t = cross(q - p, s) / cross(r, s) u = cross(q - p, r) / cross(r, s) intersect if 0 <= t <= 1 && 0 <= u <= 1

如果 cross(r, s) 接近 0，说明两条线段平行或共线，需要单独处理。

2D SAT 凸多边形检测

SAT = Separating Axis Theorem，分离轴定理。对于两个凸形状，如果存在一条轴，让两个形状投影区间不重叠，则它们不碰撞。反过来，如果所有候选轴上都重叠，则碰撞。若多边形属于某个旋转物体，先把所有本地顶点变换到世界空间。

投影重叠
存在分离轴

候选轴

2D 凸多边形的候选轴是每条边的法线。

edge = vertex[i + 1] - vertex[i] axis = normalize(perpendicular(edge)) perpendicular(x, y) = (-y, x)

数学过程

  1. 把多边形 A 和 B 的本地顶点按 worldVertex = position + R * localVertex 转到世界空间。
  2. 取世界空间多边形 A 和 B 的所有边。
  3. 对每条边求垂直轴。
  4. 把 A 和 B 的所有顶点投影到这条轴上。
  5. 得到两个区间 [minA, maxA] 和 [minB, maxB]。
  6. 如果某条轴上区间不重叠，立即返回不碰撞。
  7. 所有轴都重叠，则两个凸多边形碰撞。

3D 球体碰撞

球 - 球

球体旋转后形状不变，只有本地中心偏移会随旋转变化。本文不考虑缩放，所以半径保持不变。

worldCenterA = positionA + rotationA * localCenterA worldCenterB = positionB + rotationB * localCenterB radiusA = localRadiusA radiusB = localRadiusB
A(centerA, radiusA) B(centerB, radiusB) d = centerB - centerA collide if dot(d, d) <= (radiusA + radiusB)^2

这和 2D 圆 - 圆完全一样，只是向量从 Vec2 变成 Vec3。

球 - 平面

plane: dot(N, X) + d = 0 signedDistance = dot(N, sphere.center) + d collide if abs(signedDistance) <= sphere.radius

如果平面是地面，通常只关心球是否穿到平面下面：

penetration = radius - signedDistance collide if penetration > 0

3D AABB 碰撞

3D AABB 比 2D 多了 z 轴。三个轴都重叠才碰撞。如果物体本身有旋转，本地盒应先转为世界 OBB，再计算包住 OBB 的世界 AABB 做宽阶段；窄阶段不要继续用这个粗略 AABB 替代 OBB。

overlapX = A.min.x <= B.max.x && A.max.x >= B.min.x overlapY = A.min.y <= B.max.y && A.max.y >= B.min.y overlapZ = A.min.z <= B.max.z && A.max.z >= B.min.z collide = overlapX && overlapY && overlapZ

球 - AABB

先找 AABB 上离球心最近的点，再用球 - 点距离判断。

closest.x = clamp(center.x, min.x, max.x) closest.y = clamp(center.y, min.y, max.y) closest.z = clamp(center.z, min.z, max.z) d = center - closest collide if dot(d, d) <= radius^2

旋转盒转宽阶段 AABB

worldCenter = position + R * localCenter axisX = R * (1, 0, 0) axisY = R * (0, 1, 0) axisZ = R * (0, 0, 1) worldHalf.x = abs(axisX.x) * half.x + abs(axisY.x) * half.y + abs(axisZ.x) * half.z worldHalf.y = abs(axisX.y) * half.x + abs(axisY.y) * half.y + abs(axisZ.y) * half.z worldHalf.z = abs(axisX.z) * half.x + abs(axisY.z) * half.y + abs(axisZ.z) * half.z worldAabb.min = worldCenter - worldHalf worldAabb.max = worldCenter + worldHalf

射线检测

射线公式：

P(t) = origin + direction * t t >= 0 means in front of ray origin

射线 - 球

把射线方程代入球方程。

sphere: dot(P - C, P - C) = r^2 P = O + D * t oc = O - C a = dot(D, D) b = 2 * dot(oc, D) c = dot(oc, oc) - r^2 discriminant = b^2 - 4ac intersect if discriminant >= 0

射线 - 三角形

常用 Moller-Trumbore 算法，直接求射线在三角形重心坐标中的参数。

edge1 = v1 - v0 edge2 = v2 - v0 h = cross(rayDir, edge2) a = dot(edge1, h) if abs(a) near 0, ray parallel triangle f = 1 / a s = rayOrigin - v0 u = f * dot(s, h) q = cross(s, edge1) v = f * dot(rayDir, q) t = f * dot(edge2, q) hit if u >= 0 && v >= 0 && u + v <= 1 && t >= 0

胶囊体 = 线段扫过一个半径。它适合角色控制器，因为底部圆滑，不容易卡在台阶和边缘上。

胶囊体 = 线段 + 半径

胶囊 - 胶囊可以转化为“线段 - 线段最近距离”。

如果胶囊端点保存在本地空间，检测前先旋转和平移两个端点；本文不考虑缩放，所以半径保持不变。

OBB 与 3D SAT

OBB = Oriented Bounding Box，有方向的盒子。它比 AABB 更贴合旋转物体，但检测更复杂。

在本文的无缩放口径下，OBB 的世界半尺寸等于本地半尺寸，旋转只改变中心和方向轴。

worldCenter = position + R * localCenter worldAxisX = R * localAxisX worldAxisY = R * localAxisY worldAxisZ = R * localAxisZ worldHalfExtents = localHalfExtents

2D OBB

2D OBB 可以当成 4 个顶点的凸多边形，用 SAT 检测。候选轴是两个盒子的局部 x/y 轴，共 4 条轴。

3D OBB

3D OBB 的 SAT 候选轴共有 15 条：

A 的 3 个轴: A0, A1, A2 B 的 3 个轴: B0, B1, B2 9 个叉乘轴: cross(Ai, Bj), i = 0..2, j = 0..2 total = 3 + 3 + 9 = 15

只要任意一条轴上投影不重叠，就没有碰撞。所有轴都重叠，则 OBB 碰撞。

3D OBB SAT 的代码细节比 AABB 多很多，尤其要处理接近平行的轴。实际引擎常用成熟物理库或 GJK/EPA 处理一般凸体。

GJK 概念

GJK = Gilbert-Johnson-Keerthi，是检测任意两个凸形状是否相交的通用算法。它不需要枚举所有面轴，而是通过 Minkowski Difference 和 Support Function 迭代判断原点是否在差集内部。

Minkowski Difference

A - B = { a - b | a in A, b in B } If origin is inside A - B, then A and B intersect.

Support Function

support(shape, direction) = point in shape with max dot(point, direction) support(A - B, d) = support(A, d) - support(B, -d)

GJK 适合圆/球、胶囊、盒子、凸包等凸形状。要得到穿透深度和法线，通常再接 EPA。

学习阶段可以先掌握 SAT 和最近点算法。GJK/EPA 比较抽象，适合在你已经熟悉各种基础几何碰撞后再实现。

C++ 示例代码

下面代码偏教学版，重点是算法逻辑。实际引擎中还要处理浮点误差、接触点、连续碰撞、层级过滤、物理材质和解算器。

#include <algorithm>
#include <array>
#include <cmath>
#include <vector>

constexpr float EPSILON = 1e-6f;

struct Vec2 {
    float x = 0.0f;
    float y = 0.0f;
};

struct Vec3 {
    float x = 0.0f;
    float y = 0.0f;
    float z = 0.0f;
};

inline Vec2 operator+(Vec2 a, Vec2 b) { return { a.x + b.x, a.y + b.y }; }
inline Vec2 operator-(Vec2 a, Vec2 b) { return { a.x - b.x, a.y - b.y }; }
inline Vec2 operator*(Vec2 v, float s) { return { v.x * s, v.y * s }; }

inline Vec3 operator+(Vec3 a, Vec3 b) { return { a.x + b.x, a.y + b.y, a.z + b.z }; }
inline Vec3 operator-(Vec3 a, Vec3 b) { return { a.x - b.x, a.y - b.y, a.z - b.z }; }
inline Vec3 operator*(Vec3 v, float s) { return { v.x * s, v.y * s, v.z * s }; }

inline float Dot(Vec2 a, Vec2 b) { return a.x * b.x + a.y * b.y; }
inline float Dot(Vec3 a, Vec3 b) { return a.x * b.x + a.y * b.y + a.z * b.z; }
inline float Cross(Vec2 a, Vec2 b) { return a.x * b.y - a.y * b.x; }

inline Vec3 Cross(Vec3 a, Vec3 b)
{
    return {
        a.y * b.z - a.z * b.y,
        a.z * b.x - a.x * b.z,
        a.x * b.y - a.y * b.x
    };
}

inline float Clamp(float value, float minValue, float maxValue)
{
    return std::max(minValue, std::min(value, maxValue));
}

inline float LengthSquared(Vec2 v) { return Dot(v, v); }
inline float LengthSquared(Vec3 v) { return Dot(v, v); }

struct Transform2D {
    Vec2 position;
    float rotationRadians = 0.0f;
};

inline Vec2 Rotate(Vec2 v, float radians)
{
    float c = std::cos(radians);
    float s = std::sin(radians);
    return {
        v.x * c - v.y * s,
        v.x * s + v.y * c
    };
}

inline Vec2 TransformPoint(Transform2D transform, Vec2 localPoint)
{
    return transform.position + Rotate(localPoint, transform.rotationRadians);
}

inline Vec2 TransformDirection(Transform2D transform, Vec2 localDirection)
{
    return Rotate(localDirection, transform.rotationRadians);
}

struct RotationBasis3D {
    Vec3 axisX = { 1.0f, 0.0f, 0.0f };
    Vec3 axisY = { 0.0f, 1.0f, 0.0f };
    Vec3 axisZ = { 0.0f, 0.0f, 1.0f };
};

struct Transform3D {
    Vec3 position;
    RotationBasis3D rotation;
};

inline Vec3 RotateVector(Transform3D transform, Vec3 localVector)
{
    return transform.rotation.axisX * localVector.x +
           transform.rotation.axisY * localVector.y +
           transform.rotation.axisZ * localVector.z;
}

inline Vec3 TransformPoint(Transform3D transform, Vec3 localPoint)
{
    return transform.position + RotateVector(transform, localPoint);
}

inline Vec2 ClosestPointOnSegment(Vec2 p, Vec2 a, Vec2 b)
{
    Vec2 ab = b - a;
    float denom = Dot(ab, ab);
    if (denom < EPSILON) {
        return a;
    }

    float t = Dot(p - a, ab) / denom;
    t = Clamp(t, 0.0f, 1.0f);
    return a + ab * t;
}

inline Vec3 ClosestPointOnSegment(Vec3 p, Vec3 a, Vec3 b)
{
    Vec3 ab = b - a;
    float denom = Dot(ab, ab);
    if (denom < EPSILON) {
        return a;
    }

    float t = Dot(p - a, ab) / denom;
    t = Clamp(t, 0.0f, 1.0f);
    return a + ab * t;
}

bool Intersects(Segment2D a, Segment2D b, Vec2* hitPoint = nullptr)
{
    Vec2 p = a.a;
    Vec2 r = a.b - a.a;
    Vec2 q = b.a;
    Vec2 s = b.b - b.a;

    float rxs = Cross(r, s);
    if (std::abs(rxs) < EPSILON) {
        return false; // 平行或共线；教学版这里省略共线重叠处理。
    }

    float t = Cross(q - p, s) / rxs;
    float u = Cross(q - p, r) / rxs;

    bool hit = t >= 0.0f && t <= 1.0f && u >= 0.0f && u <= 1.0f;
    if (hit && hitPoint) {
        *hitPoint = p + r * t;
    }
    return hit;
}

2D 凸多边形 SAT

struct Projection {
    float min = 0.0f;
    float max = 0.0f;
};

Projection ProjectPolygon(const std::vector<Vec2>& vertices, Vec2 axis)
{
    Projection result;
    result.min = result.max = Dot(vertices[0], axis);

    for (size_t i = 1; i < vertices.size(); ++i) {
        float p = Dot(vertices[i], axis);
        result.min = std::min(result.min, p);
        result.max = std::max(result.max, p);
    }

    return result;
}

bool Overlaps(Projection a, Projection b)
{
    return a.max >= b.min && b.max >= a.min;
}

Vec2 Normalize(Vec2 v)
{
    float len2 = LengthSquared(v);
    if (len2 < EPSILON) {
        return { 0.0f, 0.0f };
    }

    float invLen = 1.0f / std::sqrt(len2);
    return v * invLen;
}

bool IntersectsConvexPolygonSAT(const std::vector<Vec2>& a, const std::vector<Vec2>& b)
{
    auto testAxes = [](const std::vector<Vec2>& source, const std::vector<Vec2>& a, const std::vector<Vec2>& b) {
        for (size_t i = 0; i < source.size(); ++i) {
            Vec2 p0 = source[i];
            Vec2 p1 = source[(i + 1) % source.size()];
            Vec2 edge = p1 - p0;
            Vec2 axis = Normalize({ -edge.y, edge.x });

            Projection pa = ProjectPolygon(a, axis);
            Projection pb = ProjectPolygon(b, axis);
            if (!Overlaps(pa, pb)) {
                return false;
            }
        }
        return true;
    };

    return testAxes(a, a, b) && testAxes(b, a, b);
}

bool RaySphere(Ray ray, Sphere sphere, float* outT = nullptr)
{
    Vec3 oc = ray.origin - sphere.center;
    float a = Dot(ray.direction, ray.direction);
    float b = 2.0f * Dot(oc, ray.direction);
    float c = Dot(oc, oc) - sphere.radius * sphere.radius;
    float discriminant = b * b - 4.0f * a * c;

    if (discriminant < 0.0f) {
        return false;
    }

    float t = (-b - std::sqrt(discriminant)) / (2.0f * a);
    if (t < 0.0f) {
        t = (-b + std::sqrt(discriminant)) / (2.0f * a);
    }

    if (t < 0.0f) {
        return false;
    }

    if (outT) {
        *outT = t;
    }
    return true;
}

射线 - 三角形

bool RayTriangle(Ray ray, Vec3 v0, Vec3 v1, Vec3 v2, float* outT = nullptr)
{
    Vec3 edge1 = v1 - v0;
    Vec3 edge2 = v2 - v0;
    Vec3 h = Cross(ray.direction, edge2);
    float a = Dot(edge1, h);

    if (std::abs(a) < EPSILON) {
        return false;
    }

    float f = 1.0f / a;
    Vec3 s = ray.origin - v0;
    float u = f * Dot(s, h);
    if (u < 0.0f || u > 1.0f) {
        return false;
    }

    Vec3 q = Cross(s, edge1);
    float v = f * Dot(ray.direction, q);
    if (v < 0.0f || u + v > 1.0f) {
        return false;
    }

    float t = f * Dot(edge2, q);
    if (t < 0.0f) {
        return false;
    }

    if (outT) {
        *outT = t;
    }
    return true;
}

胶囊检测

float DistanceSquaredPointSegment(Vec3 p, Vec3 a, Vec3 b)
{
    Vec3 closest = ClosestPointOnSegment(p, a, b);
    return LengthSquared(p - closest);
}

bool Intersects(Sphere sphere, Capsule capsule)
{
    float radiusSum = sphere.radius + capsule.radius;
    float dist2 = DistanceSquaredPointSegment(sphere.center, capsule.a, capsule.b);
    return dist2 <= radiusSum * radiusSum;
}

float DistanceSquaredSegmentSegment(Vec3 p1, Vec3 q1, Vec3 p2, Vec3 q2)
{
    Vec3 d1 = q1 - p1;
    Vec3 d2 = q2 - p2;
    Vec3 r = p1 - p2;

    float a = Dot(d1, d1);
    float e = Dot(d2, d2);
    float f = Dot(d2, r);

    float s = 0.0f;
    float t = 0.0f;

    if (a <= EPSILON && e <= EPSILON) {
        return LengthSquared(p1 - p2);
    }

    if (a <= EPSILON) {
        s = 0.0f;
        t = Clamp(f / e, 0.0f, 1.0f);
    } else {
        float c = Dot(d1, r);
        if (e <= EPSILON) {
            t = 0.0f;
            s = Clamp(-c / a, 0.0f, 1.0f);
        } else {
            float b = Dot(d1, d2);
            float denom = a * e - b * b;

            if (denom != 0.0f) {
                s = Clamp((b * f - c * e) / denom, 0.0f, 1.0f);
            }

            t = (b * s + f) / e;

            if (t < 0.0f) {
                t = 0.0f;
                s = Clamp(-c / a, 0.0f, 1.0f);
            } else if (t > 1.0f) {
                t = 1.0f;
                s = Clamp((b - c) / a, 0.0f, 1.0f);
            }
        }
    }

    Vec3 c1 = p1 + d1 * s;
    Vec3 c2 = p2 + d2 * t;
    return LengthSquared(c1 - c2);
}

bool Intersects(Capsule a, Capsule b)
{
    float radiusSum = a.radius + b.radius;
    float dist2 = DistanceSquaredSegmentSegment(a.a, a.b, b.a, b.b);
    return dist2 <= radiusSum * radiusSum;
}

调试建议

  • 先画出世界空间碰撞体线框，不要只看渲染模型，也不要只看本地碰撞体。
  • 确认调试线框只应用 position + rotation，不应用 scale。
  • 先实现圆/球、AABB，再实现胶囊、SAT、GJK。
  • 所有方向向量参与 dot/cross 前确认是否 normalize。
  • 比较距离时优先用距离平方，避免无意义开方。
  • 给每个碰撞检测写独立测试，例如“刚好接触”“完全分离”“一个包含另一个”。
  • 处理浮点误差，尤其是线段平行、射线贴着三角形、OBB 轴接近平行。
  • 窄阶段返回 bool 只是第一步，后续还需要 normal、depth、contact point 才能做物理解算。
  • 动态高速物体会穿透，需要连续碰撞检测 CCD，例如 swept sphere / swept AABB。

推荐学习顺序：Circle/Sphere -> AABB -> Segment/Ray -> Capsule -> 2D SAT -> 3D OBB SAT -> GJK/EPA -> Contact Solver。
