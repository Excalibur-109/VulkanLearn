Repeat(value, length) 如何让数值回绕

动画中的点在原始数轴上持续向右移动。每穿过一个完整周期，底部结果就跳回 0，但始终落在 [0, length)。拖动“周期位置”可直接查看负数如何映射。

公式如何工作

Repeat(value, length) = value − floor(value / length) × length，length > 0

先用 value / length 算出当前位置跨越了多少个周期；再用 floor 向下取整，得到必须完整移除的周期数；最后减去这些完整周期，余下部分就必然位于 [0, length)。

1. 原始位置

value 可以是任意正数或负数。图上方的点不会回绕，它代表真实、连续增长的时间、角度或 UV 偏移。

2. 选择周期

floor(value / length) 给出当前点位于哪个完整区间。负数必须向负无穷方向取整，而不是朝零截断。

3. 保留余量

减去完整周期后，剩下的局部位置就是结果。它从不等于 length，到达右端时会立刻回到 0。

为什么负数要用 floor

令 value = -0.25、length = 1：

-0.25 − floor(-0.25 / 1) × 1 = -0.25 − (-1) = 0.75
这与 std::fmod 的行为不同。

std::fmod(-0.25, 1.0) 返回 -0.25，余数保留被除数的符号；Repeat(-0.25, 1.0) 返回 0.75，结果始终落在正向循环区间。后者更适合 UV、角度和动画相位。
